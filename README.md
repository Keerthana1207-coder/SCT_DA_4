# SkillCraft Task 4 - Business Insights Report (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a marketing campaign dataset using Microsoft Power BI. The dashboard provides insights into campaign performance, ROI, acquisition cost, conversion rate, and marketing effectiveness.


## 🎯 Objective
Analyze marketing campaign performance and recommend business insights using interactive visualizations instead of building a machine learning model.

## 📊 Dashboard Features

- KPI Cards
  - Total Campaigns
  - Average ROI
  - Average Acquisition Cost
  - Average Conversion Rate
  - Total Clicks

- ROI Analysis by Marketing Channel

- Acquisition Cost by Marketing Channel

- Conversion Rate by Campaign Type

- ROI by Location

- Campaign Type Distribution

- Interactive Filters
  - Campaign Type
  - Channel Used
  - Customer Segment
  - Location

## 📈 Key Insights

- Total Campaigns: **200,005**
- Average ROI: **5.00**
- Average Acquisition Cost: **12,504**
- Average Conversion Rate: **0.08**
- Total Clicks: **109,957,667**


## 💡 Business Recommendations

- Continue investing in high-performing marketing channels.
- Improve customer targeting to increase conversion rates.
- Optimize acquisition costs for better ROI.
- Use interactive filters to analyze campaign performance across different customer segments and locations.

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Visualization
